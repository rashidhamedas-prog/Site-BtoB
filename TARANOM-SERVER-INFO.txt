 4. تنظیم DNS — مهم‌ترین قدم | DNS SETUP (CRITICAL)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  گزینه توصیه‌شده: استفاده از Cloudflare (رایگان + CDN)

  ─── مرحله الف: ثبت در Cloudflare ──────────────────────────────────────────
  1. به سایت cloudflare.com بروید و وارد حساب خود شوید
  2. روی "Add a Site" کلیک کنید
  3. دامنه  poshaktaranom.com  را وارد کنید
  4. پلن رایگان (Free) را انتخاب کنید
  5. Cloudflare دو nameserver به شما می‌دهد. مثلاً:
       NS1:  hazel.ns.cloudflare.com
       NS2:  ivan.ns.cloudflare.com
     (نام‌ها برای هر حساب متفاوت است — از همان صفحه Cloudflare بگیرید)

  ─── مرحله ب: وارد کردن Nameserver در ثبت‌کننده دامنه ─────────────────────
  6. وارد پنل ثبت دامنه‌تان شوید (nic.ir یا هر رجیستری)
  7. DNS 1 و DNS 2 را با nameserverهایی که Cloudflare داد عوض کنید
  8. ذخیره کنید — انتشار DNS تا 24 ساعت طول می‌کشد

  ─── مرحله ج: ثبت A Records در Cloudflare ──────────────────────────────────
  9. در Cloudflare، بخش DNS > Records بروید
  10. این سه رکورد را اضافه کنید:

       Type   Name                    Content         Proxy
       A      poshaktaranom.com       5.75.200.102    OFF (DNS only)
       A      www                     5.75.200.102    OFF (DNS only)
       A      api                     5.75.200.102    OFF (DNS only)

     ⚠️  مهم: Proxy را خاموش بگذارید (ابر نارنجی = OFF) تا SSL سرور کار کند

  ─── مرحله د: فعال‌سازی SSL (بعد از انتشار DNS) ───────────────────────────
  11. بعد از اینکه DNS منتشر شد، به سرور متصل شوید:
      ssh -i "C:\Users\DayaTech\.ssh\wholesale_server" wholesale-admin@5.75.200.102

  12. دستورات SSL را اجرا کنید (فایل راهنما روی سرور موجود است):
      cat /opt/taranom/SSL-SETUP.md

  یا این دستورها را مستقیم اجرا کنید:
      sudo apt-get install -y certbot
      sudo certbot certonly --webroot \
        -w /var/lib/docker/volumes/taranom_certbot_www/_data \
        -d poshaktaranom.com -d www.poshaktaranom.com -d api.poshaktaranom.com \
        --email rashidhamedas@gmail.com --agree-tos --non-interactive
      sudo mkdir -p /opt/taranom/nginx/ssl
      sudo cp /etc/letsencrypt/live/poshaktaranom.com/fullchain.pem /opt/taranom/nginx/ssl/
      sudo cp /etc/letsencrypt/live/poshaktaranom.com/privkey.pem /opt/taranom/nginx/ssl/
      sudo cp /opt/taranom/nginx/nginx.conf.ssl-ready /opt/taranom/nginx/nginx.conf
      sudo docker restart taranom_nginx

  cd /opt/taranom && sudo docker compose restart

  sudo docker exec -it taranom_postgres psql -U taranom -d taranom_db

  cd /opt/taranom && sudo bash deploy-update.sh

  # بررسی لاگ فایروال
  sudo ufw status verbose

  # ریبوت سرور (همه سرویس‌ها auto-start دارند)
  sudo reboot
